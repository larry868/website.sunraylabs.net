Go packages

All go packages can be found in the official go repository pkg.go.dev.

stockchart HTML5 stock chart package in go. Generate interactive, responsive, and performant chart with HTML5 canvas and web assembly. Experimental use of webapi package to write webassembly code in go.

timeline package provides timeslice and timemask primitives and an extention to the time.Duration struct. TimeSlice represents a range of times bounded by two dates (time.Time) From and To. It accepts infinite boundaries (zero times) and can be chronological or anti-chronological. TimeMask is used for scanning a TimeSlice and to get the time corresponding to a rounding o’clock period.

datarange package represents a range bounded by a low and a high value.

rgb package provides a uint32 RGBA color type, with a set of methods to manipulate the color. RGB type is a 32bits color: 24bits for the RGB color itself and 8 bits for the alpha channel (transparency/opacity)

verbose package usefull to generate formatted output when verbose mode is turned on.
